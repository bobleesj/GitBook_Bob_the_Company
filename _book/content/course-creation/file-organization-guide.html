URL Guide for Documenting

Look at the how the folders are managed through inspecting Learn Swift with Bob Github and https://learnswiftwithbob.com

The name of the sub-folder should be organic

intermediate-swift/delegate-datasource (Preferred) 2000_intermediate_swift/2005_swift_fundamentals (bad)

For search engine optimization, URL should be human readable and - is favored than _ for spacing between words.
